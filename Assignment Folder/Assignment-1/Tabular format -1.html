HTML Text Formatting Tags

Tag Description Example Usage Example
<h1> Creates a large heading <h1>Heading 1</h1>

Heading 1
<h6> Creates the smallest heading <h6>Heading 6</h6>
Heading 6
<p> Creates a paragraph <p>This is a paragraph.</p>

This is a paragraph.
<strong> Strongly emphasizes text <p><strong>This is a strongly emphasized text.</strong></p>

This is a strongly emphasized text.
<em> Emphasizes text <p><em>This is an emphasized text.</em></p>

This is an emphasized text.
<mark> Highlights text <p><mark>This is highlighted text.</mark></p>

This is highlighted text.
<small> Creates a smaller text <p><small>This is a smaller text.</small></p>

This is a smaller text.
<del> Strikes through text <p><del>This is a deleted text.</del></p>

This is a deleted text.
<ins> Underlines text <p><ins>This is an inserted text.</ins></p>

This is an inserted text.
<sub> Creates a subscript text <p>X<sub>2</sub> = 4</p>

X2 = 4
<sup> Creates a superscript text <p>H<sup>2</sup>O = 2002</p>

H2O = 2002
<abbr> Defines an abbreviation or acronym <p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1945.</p>

The WHO was founded in 1945.

<dfn> Defines the defining instance of a term <p>The <dfn>World Health Organization</dfn> was founded in 1945.</p>

The World Health Organization was founded in 1945.
<code> Defines a piece of computer code <p>The <code>function</code> keyword defines a function.</p>

The function keyword defines a function.
<var> Defines a variable <p>The variable <var>x</var> is used in the formula <var>x</var>².</p>

The variable x is used in the formula x².
<samp> Defines sample output from a computer program <p>The <samp>printf</samp> function is used to print text.</p>

The printf function is used to print text.
